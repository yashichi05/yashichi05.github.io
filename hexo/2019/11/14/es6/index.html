宣告

let,const,var
const 物件可以變裡面的值，不能變結構
let 只作用在宣告的{}裡面

箭頭函式

const func = (x) => x + 1 等同於 const func = function (x) { return x + 1 }

展開陣列

a = [1,2,3,4]

…a = 1,2,3,4

這可以用在 function run(…data){} ,function run2(data_main,…data){}
run(1,2,3) or run(1,2)

賦值

let family = [1,2,3,4,5]
let [a,b,c] = family
則a會是1 b是2 c是3 後面被捨棄

let data = {a:”1”,b:”2”}
let{a:aa} = data
a的名稱會變成aa 值為1

let [,data] = [1,2]
data 值為 2

縮寫

const data = 1
a = {data} 等同於 a = {data:data}
a = {def(){return}} 等同於 a = {def:function(){}}

字串

`你好 ${data}` 等同於 “你好”+data

陣列操作
foreach

array.foreach(funciton(item,index,thisArray){})

map 類似 foreach 但他一定會return ，沒指定會是undefined

array.map(funciton(item,index,thisArray){retrun})

filter 條件為true 回傳原值

array.filter(funciton(item,index){retrun true})

find 條件為true 回傳第一個true原值

array.find(funciton(item,index){retrun true})

every 所有結果為true 則回傳TRUE

array.every(funciton(item,index){retrun true})

reduce pre會是9999，此function 會從9999加總所有陣列內值並回傳retrun

array.reduce(funciton(pre,item,index){retrun item+pre},9999)
